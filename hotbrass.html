I playtested and suggested UX changes for the alpha version of upcoming indie game Hot Brass.


Along with several other user experience researchers of varying levels of experience, I signed up for a task group led by games researcher Seb Long to evaluate the usability of Hot Brass, an indie game set for release in 2020. I conducted cognitive walkthroughs on basic functions of the game and did a blind playthrough in order to evaluate the game's tutorial and learnability. Finally, I produced a report for the development team to go over and possibly implement some changes. (You can ask to look at it when the game comes out.)
